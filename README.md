# Voicebot

Este projeto implementa um voicebot simples no Google Colab que:

- Grava áudio do usuário
- Transcreve usando Whisper
- Gera resposta com modelo Gemini
- Converte resposta em áudio (TTS)

Antes de começar, você precisa de:

- Conta Google (para usar o Colab)
- Navegador com permissão de microfone
- Uma API Key do Google Gemini

Para utlizar esse Voicebot, crie um notebook no colab e importe o código disponível em o. Crie uma API Gemini e substitua no local apropriado. Excecute o código e, quando solicitado pelo navegador, clique em "Permitir" acesso ao microfone.
